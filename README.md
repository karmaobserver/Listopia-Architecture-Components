# Listopia-Android-Architecture-Components

This project is for master thesis researching. Emphasis is on [Android Architecture Componenents][arch].
The app uses a Model-View-ViewModel (MVVM) architecture for the presentation layer.


### Libraries
* [Android Support Library][support-lib]
* [Android Architecture Components][arch]
* [Android Data Binding][data-binding]
* [Dagger 2][dagger2] for dependency injection
* [Retrofit][retrofit] for REST api communication
* [Glide][glide] for image loading
* [Timber][timber] for logging
* [espresso][espresso] for UI tests
* [mockito][mockito] for mocking in tests
